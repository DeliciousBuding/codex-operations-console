# 历史归档

这个目录保存 `codex-operations-console` 的阶段性报告、优化过程记录和历史验证文档。

这些文件保留的原因是便于追踪项目演进，但它们不应该成为开源读者的第一入口。

## 包含内容

- `OPTIMIZATION-SUMMARY.md`
- `P0-CRITICAL-REVIEW.md`
- `P0-VERIFICATION-REPORT.md`
- `P1-FINAL-VERIFICATION.md`
- `P1-OPTIMIZATION-REPORT.md`
- `P2-OPTIMIZATION-COMPLETE.md`
- `VISUAL-OPTIMIZATION-COMPLETE.md`
- `screenshots-cleanup-report.md`

## 使用建议

- 想了解当前项目该怎么使用或继续开发，先看 `../README.md`
- 想了解设计边界，先看 `../visual-design-system.md`
- 想了解测试流程，先看 `../browser-test-workflow.md`
- 只有在需要追踪历史优化细节时，再进入这个目录
