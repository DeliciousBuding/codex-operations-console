## 🔧 关键修复记录

### 修复 1: 卡片圆角问题
**问题**: Semi UI 多层嵌套导致圆角丢失
**解决**: 强制应用到所有层级
```css
.semi-card,
.semi-card-content,
.semi-card-body {
  border-radius: 24px !important;
}
```

### 修复 2: 卡片内部空白
**问题**: 嵌套 padding 导致多余空白
**解决**: 移除 body padding，表格自带 margin
```css
.workspace-card-body { padding: 0; }
.table-shell { margin: 24px; }
```

### 修复 3: 选中状态不明显
**问题**: 透明度过低，视觉反馈不足
**解决**: 渐变背景 + 加粗边框
```css
.table-row-selected {
  background: linear-gradient(90deg, rgba(107,138,255,0.15), ...);
  border-left: 4px solid var(--primary);
}
```

### 修复 4: 按钮层级不清
**问题**: 所有按钮权重相同
**解决**: 建立三级按钮系统
```css
.topbar-button-primary { /* 主按钮 */ }
.topbar-button-secondary { /* 次要 */ }
.topbar-button-tertiary { /* 三级 */ }
```

## 🎯 最佳实践沉淀

### 1. 设计系统原则
- **8px 基数**: 所有间距使用 8 的倍数
- **变量优先**: 使用 CSS 变量而非硬编码
- **层级清晰**: 字体大小至少相差 4px
- **避免嵌套**: 组件自己管理间距

### 2. OLED 优化原则
- **真黑背景**: #000000（非深灰）
- **降对比度**: 避免纯白文字
- **材质层次**: 清晰的灰度分层
- **微妙动画**: 0.2s ease-in-out

### 3. 测试原则
- **多模态**: 截图 + DOM + 交互
- **系统化**: 覆盖所有页面
- **文档化**: 记录问题和解决方案
- **可重复**: 标准化工作流
