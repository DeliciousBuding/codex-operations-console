# 行为准则

为了让这个项目保持专业、友好、可持续，我们希望所有参与者都遵守以下约定。🤝

## 🌱 我们鼓励的行为

- 尊重不同经验水平的贡献者
- 使用清晰、具体、建设性的表达
- 在讨论中聚焦问题本身，而不是攻击个人
- 接受反馈，也愿意解释自己的判断依据
- 对安全、可维护性和用户体验保持基本责任感

## 🚫 我们不接受的行为

- 人身攻击、羞辱、歧视、骚扰
- 恶意抬杠、反复挑衅或蓄意制造冲突
- 泄露他人隐私、敏感凭据或未公开信息
- 在讨论区发布明显与项目无关的垃圾内容

## 🛡️ 维护者职责

维护者有权对不当行为进行提醒、删除内容、关闭讨论或拒绝协作。

## 📌 适用范围

本准则适用于：

- Issue
- Pull Request
- Discussions
- 与本项目协作相关的公开交流场景
